# BLORM



## Getting Started

First, clone the repo
```bash
git clone https://github.com/blorm-network/blorm_dapp
```

Then, install the dependencies.

```bash
npm install
```

Now, run the web app:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
